Bootstrap and Masonry

Integrate Masonry with the grid system and cards component.

Masonry is not included in Bootstrap. Add it by including the JavaScript plugin manually, or using a CDN like so:

<script src="https://cdn.jsdelivr.net/npm/masonry-layout@4.2.2/dist/masonry.pkgd.min.js" async></script>

By adding data-masonry='{"percentPosition": true }' to the .row wrapper, we can combine the powers of a responsive grid and Masonry’s positioning.
