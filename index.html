Class Info

This portfolio is part of Professor Soyeon Jang’s block-plan course: Open Source Studio & Web Service Development. The assignment focuses on setting up the environment, practicing HTML/CSS, and deploying a working site through GitHub and Netlify.

Tasks

  • Set up the development environment and free hosting service
    • Install Visual Studio Code
    • Create a new project
    • Install at least 3 useful extensions (e.g. HTML snippets)
    • Connect project with Git and GitHub repository
    • Sign up on Netlify, import GitHub project, deploy, and confirm service works
    • Edit in VS Code → push to GitHub → confirm deployment on Netlify URL
  • Practice HTML/CSS
    • Use an AI tool (e.g. ChatGPT) to generate a sample developer page → save as index.html
    • Create nostyle.html by referencing W3Schools demo pages
    • Modify top title to Welcome to [Name]’s Homepage (Student ID)
    • Choose 2 stylesheet demos (style1, style2) and recreate them using CSS
    • Recreate one official Bootstrap example → save as example.html
    • Link all pages (nostyle.html, style1.html, style2.html, example.html) from index.html
    • Push all files to GitHub with version control
    • Write a README.md with the Netlify deploy URL
